> Tier A / 新規性 ★★★☆☆ / Lazy Pricesの発展形。差別化は可能だが引用必須

---

## 一言で

決算短信やセグメントデータに大きな変化があるのに、株価が反応していない企業を特定する。

## なぜやるのか

市場は全ての情報を即座に反映するわけではない。特にテキスト情報（定性情報）の反映は数値データより遅い（PEAD.txtで実証済み）。「変化しているのに気づかれていない企業」は最も有望な投資候補。

## 先行研究との関係（重要）

**Lazy Prices（Cohen et al., 2020, Journal of Finance）を必ず読んでから着手すること。**

|  | Lazy Prices | 我々のアプローチ |
| --- | --- | --- |
| 対象 | 10-K/10-Q全文 | 決算短信の定性部分 + セグメントデータ |
| 変化の測定 | テキスト全体のコサイン類似度 | LLMによる意味的変化 + セグメント構成比変化 |
| シグナル | 「変化した企業」をショート | 「変化したのに株価無反応の企業」をロング |
| 市場 | 米国 | 日本 |

## 試して欲しい技術

### Step 1: テキスト変化度の計算（3-5日）

```python
# 同一企業の前期と今期の決算短信テキストの変化度を測定
for company in all_companies:
    prev_text = get_qualitative_text(company, period="prev")
    curr_text = get_qualitative_text(company, period="curr")

    # 方法1: コサイン類似度（Lazy Pricesと同じ基本手法）
    cos_sim = cosine_similarity(embed(prev_text), embed(curr_text))
    text_change = 1 - cos_sim

    # 方法2: LLMによる意味的変化判定（我々の差別化）
    semantic_change = llm_judge_change(prev_text, curr_text)

    # 方法3: セグメント構成比の変化量
    segment_change = calc_segment_ratio_change(company)
```

### Step 2: 株価反応度の測定（2-3日）

```python
# 決算発表後の株価反応を測定
for company in all_companies:
    announcement_date = get_announcement_date(company)

    # CAR（累積異常リターン）を計算
    car = calc_cumulative_abnormal_return(
        company,
        window=(-1, +5),  # 発表前日〜発表後5日
        benchmark="TOPIX"
    )

    price_reaction = abs(car)  # 反応の大きさ
```

### Step 3: マトリクス化（1-2日）

```
          株価反応: 小          株価反応: 大
変化: 大  【★投資候補★】        織り込み済み
変化: 小  変化なし（無視）        過剰反応
```

左上の「変化大×株価反応小」が我々のターゲット。

### Step 4: バックテスト（1-2週間）

「静かな変化」として検知した企業のN日後/N月後のリターンを検証。

## 最大のリスク

- Lazy Pricesとの差別化が不十分だと「二番煎じ」と見られる
- 株価データの取得・処理が追加で必要（現在のスキルにはない）
- 日本市場特有の流動性の問題（小型株は出来高が少なく、反応が遅いのが「普通」の場合がある）
